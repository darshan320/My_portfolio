Adaptive Smart Fan Controller

IoT / Embedded
Overview

A closed-loop temperature control system built around an ESP32 microcontroller. The system continuously samples ambient temperature and drives a PWM-controlled fan through a tuned PID algorithm, holding the target temperature within a tight tolerance band regardless of changing ambient conditions or load.

Approach
  • Implemented a PID control loop in firmware to dynamically adjust fan PWM duty cycle based on real-time temperature error.
  • Built a BLE telemetry layer to stream live temperature, fan speed, and system status to a companion mobile dashboard.
  • Added Over-The-Air (OTA) firmware update support, allowing the controller logic to be updated remotely without re-flashing over USB.
  • Tuned PID gains experimentally to minimize overshoot while keeping response time fast under sudden thermal load changes.
Outcome

The final system holds temperature within ±0.5°C of the target setpoint, automatically adapting fan speed in real time, while remaining fully maintainable in the field through OTA updates.

02 / ML Research
🧠

Ensemble TinyML Classifier on Edge

ML Research
Overview

An ensemble classifier combining a decision-tree model and a compact neural network, compressed and deployed directly onto an STM32 microcontroller for fully on-device gesture recognition — no cloud inference required.

Approach
  • Trained and compared decision-tree and neural-network models on a 6-class gesture dataset, then combined them into an ensemble to improve robustness.
  • Used Edge Impulse to handle data pipeline management, model optimization, and quantization for microcontroller deployment.
  • Converted the trained model to TensorFlow Lite format to minimize memory footprint and inference latency on constrained hardware.
  • Profiled inference time and RAM usage directly on the STM32 to validate real-time feasibility.
Outcome

The deployed ensemble reaches 94.2% classification accuracy across all 6 gesture classes, with inference completing in under 15ms and a total RAM footprint of just 64KB — well within the constraints of the target microcontroller.
